Here are some of my projects

  • Jobly

    Javascript ES6, React, Express, and PostgreSQL

    Jobly is a full stack web application of a mock job board site where users can sign up and login. Logged in users gain access to a list of companies with search and filtering capabilities. Each company has a list of job postings that a user can view. A user may apply to a job posting, but each user is restricted from duplicate applications to a single job posting posted by a company.

  • Warbler (Twitter Clone)

    Python, Flask, PostgreSQL

    Created a Full-stack Twitter clone including features for posting messages, liking posts, following users, handling account details, and login/logout authentication. Added user authentication/authorization to RESTful API routes utilizing Bcrypt password hashing. Managed many-to-many database relationships with Python ORM SQLAlchemy.

  • Final Frontier NASA API

    JavaScript ES6, React, HTML, CSS/Bootstrap
